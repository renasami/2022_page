# 今回必要な、（知ってると便利な)基本コマンド集

## linux　コマンド
* `cd` 		作業ディレクトリ移動
* `ls`		作業ディレクトリ内のファイル確認
* `pwd`		現在のディレクトリのパスを表示
* `mkdir`		ディレクトリ作成
* `touch`		ファイルの作成

## dockerコマンド
* `docker ps ` dockrコンテナの一覧を表示するためのコマンド
* `docker rmi` イメージを削除するためのコマンド
* `docker compose up` docker-compose.ymlの構成を全て起動するためのコマンド
